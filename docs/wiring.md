# 🔌 接線圖說明

## 硬體清單

| 元件 | 數量 | 說明 |
|------|------|------|
| Arduino Uno | 1 | 或相容開發板 |
| DHT11 或 DHT22 | 1 | 溫濕度感測器 |
| 杜邦線 | 3 | 母對公 |
| USB 傳輸線 | 1 | Type-A 對 Type-B |

## 接線方式

### DHT11/22 感測器接線

```
                    ┌─────────────────┐
                    │   DHT11/22      │
                    │   ┌───────┐     │
                    │   │ ░░░░░ │     │
                    │   └───────┘     │
                    │   1   2   3   4 │
                    └───┼───┼───┼───┼─┘
                        │   │   │   │
                        │   │   │   └── (不使用)
                        │   │   │
                        │   │   └────── GND (接地)
                        │   │
                        │   └────────── DATA (資料)
                        │
                        └────────────── VCC (電源)
```

### Arduino Uno 連接

```
┌──────────────────────────────────────────────┐
│                 ARDUINO UNO                   │
│                                              │
│  ┌────────────────────────────────────────┐  │
│  │ DIGITAL (PWM~)                         │  │
│  │ 13 12 11 10  9  8  7  6  5  4  3  2    │  │
│  └────────────────────────────────────────┘  │
│                                              │
│  ┌──────────┐          ┌──────────────────┐  │
│  │   USB    │          │ ANALOG IN        │  │
│  │  Type-B  │          │ A0 A1 A2 A3 A4 A5│  │
│  └──────────┘          └────────────────┬─┘  │
│         │                               │    │
│         │  ┌────────────────────────────┘    │
│  ┌──────┴──┴────┐                           │
│  │ POWER        │    DHT DATA ──────────────┤ A5
│  │ 5V  GND  ... │                           │
│  └──┬───┬───────┘                           │
│     │   │                                   │
│     │   │    DHT GND ───────────────────────┤ GND
│     │   │                                   │
│     │        DHT VCC ───────────────────────┤ 5V
│     │                                       │
└─────┴───────────────────────────────────────┘
```

## 接線表格

| DHT 腳位 | 說明 | Arduino 腳位 | 線色建議 |
|----------|------|-------------|----------|
| 1 (VCC)  | 電源 | 5V          | 🔴 紅色 |
| 2 (DATA) | 資料 | A5          | 🟡 黃色 |
| 3 (NC)   | 不使用 | -         | -        |
| 4 (GND)  | 接地 | GND         | ⚫ 黑色 |

> **注意**：有些 DHT 模組只有 3 支腳位（VCC、DATA、GND），這種模組的接線更簡單。

## 實際接線照片

```
        DHT11/22
        ┌─────┐
        │     │
        │░░░░░│
        └─┬─┬─┬─┘
          │ │ │
    ┌─────┘ │ └─────┐
    │       │       │
   VCC    DATA    GND
    │       │       │
    │       │       │
    ▼       ▼       ▼
 Arduino  Arduino  Arduino
   5V       A5      GND
```

## 注意事項

1. **電源電壓**
   - DHT11 工作電壓：3.3V - 5.5V
   - DHT22 工作電壓：3.3V - 5.5V
   - 建議使用 Arduino 的 5V 輸出

2. **資料腳位**
   - 程式碼預設使用 A5
   - 如需更改，請修改 `dht_sensor.ino` 中的 `DHTPIN`

3. **上拉電阻**
   - 部分 DHT 模組內建上拉電阻
   - 如果讀取不穩定，可在 DATA 與 VCC 之間加 10K 電阻

4. **接線距離**
   - 建議線長不超過 20 公分
   - 過長可能導致訊號不穩定

## DHT11 vs DHT22 差異

| 規格 | DHT11 | DHT22 |
|------|-------|-------|
| 溫度範圍 | 0~50°C | -40~80°C |
| 溫度精度 | ±2°C | ±0.5°C |
| 濕度範圍 | 20~90% | 0~100% |
| 濕度精度 | ±5% | ±2% |
| 取樣速度 | 1 Hz | 0.5 Hz |
| 價格 | 較便宜 | 較貴 |

## 如何確認感測器型號

1. 查看感測器上的印刷標籤
2. DHT11 通常是藍色外殼
3. DHT22 通常是白色外殼（也可能標示為 AM2302）
